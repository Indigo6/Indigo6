Hugo建主页之路(2): 文章字数、阅读时间和阅读量统计

方林 2021.7.24   |   共411字 ，约 1 分钟   |  阅读

字数和阅读时间统计

  1. 通过检查网页，找到文章标题、作者所处位置

  2. 用VS Code打开代码，搜索检查结果

  3. 加入以下代码

    <span>&nbsp; | &nbsp; 共{{ .WordCount  }}字</span>，
约<span> {{ .ReadingTime  }} 分钟</span>

  4. 统计字数不正确

    一打开 Chrome 懵逼了。我辛辛苦苦码了这么久才 69 个字？？？你豁鬼哦！！（╯‵□′）╯︵┴─┴

    Google 了一下才知道，原来 Hugo 默认不会统计中文 (详情)，需要在 config.toml 中添加这么一句配置才可以开启对中文的统计

    hasCJKLanguage = true

    Copy

    正常了
